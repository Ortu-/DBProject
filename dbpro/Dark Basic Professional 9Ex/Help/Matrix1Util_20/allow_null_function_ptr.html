Matrix1 Utility Plug-ins

ALLOW NULL FUNCTION PTR


Syntax

ALLOW NULL FUNCTION PTR Flag

Description

Be default, the plug-in will trigger an error for any calls where the function pointer is set to 0. You can use this flag to make these calls cause the function call to be ignored instead.

Note that setting or unsetting this flag does not affect the results given by the FUNCTION PTR IS VALID ( ) function.
