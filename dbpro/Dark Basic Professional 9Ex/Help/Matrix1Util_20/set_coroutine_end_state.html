SET COROUTINE END STATE


Syntax

SET COROUTINE END STATE Coroutine ID, Active State

Description

When a new coroutine is created, it is set to be inactive when the coroutine function ends. By using this command, the coroutine can be set to automatically restart at this point instead, and so will never become inactive.

States currently available are:
- 0 = Switch to inactive when Coroutine function ends and return control to the main process (default setting).
- 1 = Restart the Coroutine function when it ends.
