TG SET GIZMO SIZE gizmoID , size#

Sets the display size of the gizmo.The default size is 1.0 if you set this value to 0.5 then the size of the gizmo will be halved , or if you set to 2.0 the size will be doubled ...etc...

parameters :
gizmoID - The id of the gizmo you want to set
size# - a float size value
