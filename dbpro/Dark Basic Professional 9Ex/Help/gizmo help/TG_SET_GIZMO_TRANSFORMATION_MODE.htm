TG SET GIZMO TRANSFORM MODE gizmoID , mode

Sets the transform mode of the gizmo.You can choose from 3 presets :
1 - The transformation of the gizmo is Local
2 - The transformation of the gizmo is World
3 - The transformation of the gizmo is View
In most cases the rotate gizmo should be set to 1 , the move gizmo to 2 , and the scale to 1

NOTE : To maintain compatibility and simplicity , the scale is ALWAYS calculated in World space.The Mode flag will change only it's alignment

parameters :
gizmoID - The id of the gizmo you want to set
mode - see desricption above...
