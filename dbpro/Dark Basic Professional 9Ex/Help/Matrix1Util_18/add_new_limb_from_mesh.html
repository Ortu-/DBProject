ADD NEW LIMB FROM MESH


Syntax

ADD NEW LIMB FROM MESH MeshId
LimbId = ADD NEW LIMB FROM MESH ( MeshId )

Description

This command adds a new limb to the complex object. The mesh data for the limb will be generated from the mesh specified.

The function form of this command will return a limb id that can be used by a later limb to set as a parent limb. This id will become useless after the object has been finalised and should not be used beyond that point.
