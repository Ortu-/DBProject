MAKE OBJECT FROM BANK


Syntax

MAKE OBJECT FROM BANK Object number, Bank number
MAKE OBJECT FROM BANK Object number, Bank number, Start position

Description

The first command will take a bank containing a DBO formatted object and will create an object from that.

The second command is the same, except that you can specify the start position in the bank and the amount of data to use.
