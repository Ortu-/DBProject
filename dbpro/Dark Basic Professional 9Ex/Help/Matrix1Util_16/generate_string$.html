GENERATE STRING$


Syntax

Random String = GENERATE STRING$( Size )
Random String = GENERATE STRING$( Source String, Size )

Description

This function will a randomly generated string of upper and lowercase letters and numbers of the size specified.

In the second form of the function, the characters will be picked randomly from the list that you specify.
