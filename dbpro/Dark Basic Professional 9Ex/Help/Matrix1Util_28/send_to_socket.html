SEND TO SOCKET


Syntax

BytesWritten = SEND TO SOCKET( Handle, BufferAddress, BufferSize, IpAddress, PortNo )

Description

This command will attempt to transmit the data to the remote socket as a single datagram - it can only be used with a UDP socket.

If the socket has not already been bound to a local port using the BIND SOCKET( ) function, it will be assigned a random local port.

It is both possible and legal to transmit a zero-sized batch of data using this command.

The return value will be the number of bytes transmitted.
