Matrix1 Utility Plug-ins

Matrix1Util_03

This plug-in provides the system objects needed to synchronise with other programs. At the moment this is limited to mutex's but will also include Semaphores and Events in the future.

Synchronisation Creation

DELETE SYSOBJ 
MAKE SYSOBJ MUTEX 

Synchronisation Information

FIND FREE SYSOBJ 
GET SYSOBJ NAME$ 
RELEASE RESERVED SYSOBJ 
RESERVE FREE SYSOBJ 
SYSOBJ EXIST 
SYSOBJ RESERVED 

Locking and Unlocking

IS SYSOBJ LOCKED 
LOCK SYSOBJ 
TRY LOCK SYSOBJ 
UNLOCK SYSOBJ
