SET LIMB TEXTURE BLEND MODE


Syntax

SET LIMB TEXTURE BLEND MODE Object number, Limb number, Stage, Blend mode, Blend mode Arg1
SET LIMB TEXTURE BLEND MODE Object number, Limb number, Stage, Blend mode, Blend mode Arg1, Blend mode arg2

Description

This command is used to set the texture blending mode for the specified texture stage.

The safe blend modes are defined within the DirectX documentation and have a prefix of D3DTOP.

The safe blend arguments are defined within the DirectX documentation and have a prefix of D3DTA.
