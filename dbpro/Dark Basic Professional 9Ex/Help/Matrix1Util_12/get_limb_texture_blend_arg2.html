GET LIMB TEXTURE BLEND ARG2


Syntax

Blend Mode = GET LIMB TEXTURE BLEND ARG2( Object number, Limb number, Texture Stage )

Description

Returns the second blend mode argument of the texture stage for the limb.

The safe blend arguments are defined within the DirectX documentation and have a prefix of D3DTA.

Constants for allowable blend arguments:
#constant D3DTA_DIFFUSE = 0
#constant D3DTA_CURRENT = 1
#constant D3DTA_TEXTURE = 2
#constant D3DTA_TFACTOR = 3
#constant D3DTA_SPECULAR = 4
#constant D3DTA_TEMP = 5
#constant D3DTA_CONSTANT = 6
#constant D3DTA_COMPLEMENT = 16
#constant D3DTA_ALPHAREPLICATE = 32
