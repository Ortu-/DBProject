SET DEFAULT ERROR TITLE


Syntax

SET DEFAULT ERROR TITLE ErrorTitle$
SET DEFAULT ERROR TITLE

Description

This command can be used to override the default title for the DBPro error messagebox.

The first form of the command will force the error message title to be replaced with the text that you use.

The second form will restore the default DBPro error message title (which just says 'Error').

Note that setting the error title to the empty string is not the same as using the second form of the command, and will just result in no title being added to the message box.
