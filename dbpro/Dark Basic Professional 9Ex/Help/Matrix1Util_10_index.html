Matrix1 Utility Plug-ins

Matrix1Util_10

This plug-in allows you to write formatted logging information to a log file.

The formatting must be provided by you, but can be almost any text you can output. For example, it is possible to output HTML or RTF to the file.

Opening the log file

CLOSE LOG 
OPEN LOG 
OPEN LOG DEBUG 

Setting log tags

CLEAR ALL TAGS 
CLEAR LOG TAG 
SET LOG TAG 
SET LOG UNKNOWN TAG 

Writing to the log

WRITELN LOG
