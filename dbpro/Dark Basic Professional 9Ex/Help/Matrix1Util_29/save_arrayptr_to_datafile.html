SAVE ARRAYPTR TO DATAFILE


Syntax

SAVE ARRAY TO DATAFILE Array pointer, Datafile Id

Description

This command will cause the array to be saved into the datafile specified. It will handle arrays of both basic and UDT types.

The array must be correctly DIM'med before use or else an error will be triggered.
