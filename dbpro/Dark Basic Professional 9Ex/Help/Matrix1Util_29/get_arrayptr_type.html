GET ARRAYPTR TYPE


Syntax

Array type = GET ARRAYPTR TYPE( Array pointer )

Description

This function is identical to GET ARRAY TYPE, except that it operates upon an array pointer rather than an array.

The type returned will be one of the following:
0 = Integer
1 = Float
2 = String
3 = Boolean
4 = Byte
5 = Word
6 = Dword
7 = Double float
8 = Double integer
9+ = User defined type
