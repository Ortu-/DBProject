COPY BANK


Syntax

COPY BANK Bank Number, Start Position, Size, Target Position
COPY BANK Source Bank Number, Start Position, Size, Target Bank Number, Target Position

Description

This command allows you to copy a section of a bank to another area of the same bank, or to a completely different bank.

The first form of this command allows copying within a bank. It is safe for the source and target areas to overlap.

The second form of this command allows copying between different banks. It is safe for the source and target banks to be the same. It is also safe for the source and target areas to overlap (for example when copying between two banks mapped to the same target).
