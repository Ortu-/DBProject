WRITE BANK BOOLEAN


Syntax

WRITE BANK BOOLEAN Bank Number, Start Position, Boolean Value

Description

This command writes the boolean value to the position specified within the bank. If the value specified is non-zero, then a value of 1 will be written. Otherwise a value of 0 will be written.

A boolean takes up 1 byte of storage and holds the values 0 or 1.
