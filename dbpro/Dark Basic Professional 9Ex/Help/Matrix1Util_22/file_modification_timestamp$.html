FILE MODIFICATION TIMESTAMP$


Syntax

TimeStamp$ = FILE MODIFICATION TIMESTAMP$ ( Filename )

Description

This function will return the modification date and time of the specified file in the format CCYYMMDDhhmmss.

Note that when a file is modified using the MAP FILE TO BANK command, the timestamp of the file is NOT modified for efficiency reasons.
