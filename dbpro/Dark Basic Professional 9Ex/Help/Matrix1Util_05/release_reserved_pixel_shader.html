RELEASE RESERVED PIXEL SHADER


Syntax

RELEASE RESERVED PIXEL SHADER PixelShaderId

Description

This command is used to release a previously reserved id back into the pool so that it can be returned by FIND FREE PIXEL SHADER or RESERVE FREE PIXEL SHADER at a later point.

If the pixel shader id is actually is use, then this command will do nothing.
