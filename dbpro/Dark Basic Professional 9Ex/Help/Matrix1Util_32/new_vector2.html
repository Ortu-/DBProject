NEW VECTOR2


Syntax

VectorID = NEW VECTOR2( )
VectorID = NEW VECTOR2( x, y )

Description

Creates a new vector2 using the next free vector id, and returns that id as its result.

The second form of this command uses the values to initialise the vector.
