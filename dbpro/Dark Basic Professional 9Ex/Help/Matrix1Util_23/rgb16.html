Matrix1 Utility Plug-ins

RGB16


Syntax

16 bit Colour = RGB16 ( Red, Green, Blue )
16 bit Colour = RGB16 ( Alpha, Red, Green, Blue )

Description

These functions will generate a 16 bit colour value based upon the values specified.

Although a full alpha value can be specified, this will only take up the space of 1 bit within the returned colour. i.e., alpha will either be on or off.

All values should be specified in the range 0 to 255, as per the standard RGB function.
