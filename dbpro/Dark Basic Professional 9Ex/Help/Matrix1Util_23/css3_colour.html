CSS3 COLOUR


Syntax

Colour = CSS3 COLOUR ( ColourName$ )
Colour = CSS3 COLOUR ( ColourName$, Alpha )

Description

The World Wide Web Consortium (W3C) defines standard for the web, and has specified a set of colour names for cascading style sheets. This command allows you to specify on of the standard colour names and to get the appropriate RGB colour returned.

The first version of this command returns a colour with an alpha value of 255 (no fade) set.

The full list of official colours supported can be found in the following link: CSS Color Module Level 3
