Matrix1 Utility Plug-ins

SET ZONE


Syntax

SET ZONE Zone number, X1, Y1, X2, Y2
SET ZONE Zone number, X1, Y1, Z1, X2, Y2, Z2

Description

This command is used to create or modify the area or volume that the zone covers.

The first version of this command will create a 2D zone.

The second version of this command will create a 3D zone.
