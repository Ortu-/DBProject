TICKER RESET


Syntax

TICKER RESET Ticker
Ticks Missed = TICKER RESET ( Ticker )

Description

The command form of this instruction will simply reset the number of outstanding ticks to 0.

The function form of this instruction will do the same, except that it will return the number of ticks that were missed before the reset was applied.

These commands are for you to use when your game loop is too slow and has missed ticks, but you don't actually need to catch up with those ticks.
