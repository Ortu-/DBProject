FIRST WINDOW CLASS


Syntax

Window Handle = FIRST WINDOW CLASS ( Window Class )
Window Handle = FIRST WINDOW CLASS ( Window Class, Search Mode )

Description

The first form of this function will search for all windows registered with the class name provided and return the handle of the first found.

The second form if this function allows the class to be searched in more flexible ways. The mode should be set to one of the following values:
- 0 = Full class match, case-sensitive
- 1 = Full class match, case-insensitive
- 2 = Partial class match, case-sensitive
- 3 = Partial class match, case-insensitive

Further window handles can be retrieved by continuing to call the NEXT WINDOW function.

If no windows are found then this function will return a value of 0.
