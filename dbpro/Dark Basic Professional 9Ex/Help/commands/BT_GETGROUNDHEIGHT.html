BT GetGroundHeight


Syntax

Height=BT GetGroundHeight(TerrainID,X,Z)

Description

This gets the ground height of the specified point. The X and Z positions are in terrain space coordinates. Meaning it is not effected by external positioning and scale. Point 0,0 will always be the top left of the map and the tile size will always be equal to the terrain scale.

Parameters

TerrainID (integer) - This is the unique ID of the terrain which you want to get the height of.
X (float) - This is the X position of the point you want to get the height.
Z (float) - This is the Z position of the point you want to get the height.

Returns

The Y position of the point.
