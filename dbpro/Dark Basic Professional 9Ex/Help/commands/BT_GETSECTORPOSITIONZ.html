BT GetSectorPositionZ


Syntax

Position=BT GetSectorPositionZ(TerrainID,LODLevel,SectorID)

Description

This gets the Z coordinate of which the specified sector is located.

Parameters

TerrainID (integer) - This is the identifier of the terrain which the sector is on.
LODLevel (integer) - This is the LOD level number which the sector is on. LOD level 0 is the highest detail lod level.
SectorID (integer) - This is the idedifier of the sector which to get the position of. It starts at 1 and goes to the value returned by BT GetSectorCount.

Returns

The Z coordinate of the sector.
