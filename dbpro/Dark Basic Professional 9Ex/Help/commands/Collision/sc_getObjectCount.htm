SC_getObjectCount

SYNTAX
SC_getObjectCount ()

returns: the total number of objects checked during casting collisions
since this command was last called.

like 'getRayCount' this command gives you some idea about how
hard the collision system is working, objects that have collision turned off,
are in a different group, or are excluded are not counted.
Calling this command resets this count to 0.
