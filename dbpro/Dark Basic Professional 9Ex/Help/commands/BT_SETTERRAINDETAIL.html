BT SetTerrainDetail


Syntax

BT SetTerrainDetail(TerrainID,ImageID)

Description

This function sets the detailmap of the terrain. It can be used at any time as long as the terrain exists

Parameters

Terrain ID (integer) - This is the identifier of the terrain you want to set the texture to. This value is created in the BT MakeTerrain function.
Image ID (integer) - This is the DBPro Image ID of the detailmap.

Returns

Nothing.
