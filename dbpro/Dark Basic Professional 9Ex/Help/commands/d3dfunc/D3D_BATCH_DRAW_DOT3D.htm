D3D_BATCH_DRAW_DOT3D

This command will draw all dots in the current batch. Setting Z Enable to 1 will enable Z-Buffering. When the clear flag is set to
anything but 0, then the current batch will be deleted after use, and any further use will require reinitialisation using
D3D_BATCH_SET_DOT3D.

SYNTAX

D3D_BATCH_DRAW_DOT3D Camera, Z Enable, Clear Flag
