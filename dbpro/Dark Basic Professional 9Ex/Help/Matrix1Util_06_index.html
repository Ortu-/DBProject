Matrix1 Utility Plug-ins

Matrix1Util_06

This plug-in will allow you to raise your own DBPro errors and to control the title of the error message box when DBPro raises its own errors.

All error messages will report the line number where the error was triggered.

Error Title

SET DEFAULT ERROR TITLE 

Triggering Errors

ASSERT ERROR 
RAISE ERROR 
REPORT ERROR 

Capturing Errors

SET ERROR FUNCTION
