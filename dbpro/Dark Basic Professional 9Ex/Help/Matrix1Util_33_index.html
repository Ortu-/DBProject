Matrix1 Utility Plug-ins

Matrix1Util_33

This plug-in allows you to draw/render to images as well as bitmaps.

For the most part, the commands and functions provided suppliment the existing functionality provided by using a 'camera to image' as the render target, although several convenience functions have also been provided.

Setting draw targets

DRAW TARGET 
DRAW TARGET HEIGHT 
DRAW TARGET TYPE 
DRAW TARGET WIDTH 
DRAW TO BITMAP 
DRAW TO CAMERA 
DRAW TO IMAGE 
SET DRAW TARGET 

Image specific commands

IMAGE ALLOWS RENDER 
MAKE IMAGE 

Camera specific commands

CAMERA IMAGE
