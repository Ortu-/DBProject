DELIMIT CSV$


Syntax

Delimited String = DELIMIT CSV$( Source String )
Delimited String = DELIMIT CSV$( Source String, Force Delimiting )

Description

These functions takes the string passed and translates it into an acceptable format for use as a field within an Excel-format CSV record.

Double-quotes will be added under the following circumstances:
- The string contains one or more commas.
- The string contains one or more double-quotes.
- The string contains leading or trailing space or tab characters.
- The Force Delimiting flag on the second form of the function is set to a non-zero value.
