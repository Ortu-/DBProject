Matrix1 Utility Plug-ins

OBJECT GLUED LIMB


Syntax

Limb Number = OBJECT GLUED LIMB ( ObjectID )

Description

If the object specified is glued to another object, this function will return the limb number of the parent object that this object is glued to.

If the object is not glued, a value of zero will be returned.
