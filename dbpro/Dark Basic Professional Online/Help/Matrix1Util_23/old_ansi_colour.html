OLD ANSI COLOUR


Syntax

Colour = OLD ANSI COLOUR ( Ansi Colour ID )
Colour = OLD ANSI COLOUR ( Ansi Colour ID, Alpha )

Description

This function will return a colour based upon the original Ansi terminal colour set, without the half-bright colours.
The first version of this command returns a colour with an alpha value of 255 (no fade) set.

The numbers to use are:

0 - black
1 - red
2 - green (lime)
3 - yellow
4 - blue
5 - magenta (fuchsia)
6 - cyan
7 - white

The names in brackets are the official CSS3 colour names
