GET ARRAY ITEM SIZE


Syntax

Item size = GET ARRAY ITEM SIZE( Array() )

Description

This function will retrieve the size for all items within the array.

The size be one of the following for built-in types
1 - Boolean, Byte
2 - Word
4 - Integer, Float, String, Dword
8 - Double float, Double integer

For UDT's the size will vary depending on the types you have defined within it.
