SCREEN MODE


Syntax

Mode = SCREEN MODE( )

Description

This function will return certain values depending on the current display mode:

0 - Fullscreen Exclusive
1 - Windowed
2 - Windowed - Desktop
3 - Windowed - Full Screen
4 - Hidden

If you are using the 'Windowed - Desktop' mode, then this will be reported as fullscreen if the taskbar is set to auto-hide.
