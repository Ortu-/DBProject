DATAFILE SKIP


Syntax

DATAFILE SKIP FileId, Offset

Description

This command moves the current read/write position in the file by the specified offset.

It is quite valid to set the position past the end of the file - as soon as you write to that position the file will be extended with null bytes to that point.

If you attempt to move to a point prior to the start of the file, the read/write position will be set to the start of the file silently.
