MAP MEMORY TO BANK


Syntax

MAP MEMORY TO BANK Memory Pointer, Bank Number, Size

Description

This command will create a bank that is mapped over a block of already-allocated memory. The bank created will be of the size specified. Any modifications made to the bank will automatically be reflected within the allocated memory area.

Care must be taken to release the bank prior to freeing the memory, otherwise you may overwrite memory that is being reused by the system.
