Matrix1 Utility Plug-ins

REGISTER FUNCTION PTR


Syntax

REGISTER FUNCTION PTR Function Pointer

Description

If you have other methods to get function pointers (perhaps via callbacks, or other function calls) then this command will allow you to register that function with this plug-in, allowing you to call that function using the commands in this plug-in.

It is your responsibility to ensure that the function pointer is valid, and that the function pointer is unregistered if the function becomes unavailable (such as when unloading the DLL it is contained within).
