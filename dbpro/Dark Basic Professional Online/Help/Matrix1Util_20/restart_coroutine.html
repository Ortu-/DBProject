Matrix1 Utility Plug-ins

RESTART COROUTINE


Syntax

RESTART COROUTINE Coroutine ID

Description

Be default, when the function that was called as a coroutine ends, the coroutine is flagged as inactive, but is not deleted. Using this command on an inactive coroutine will cause the coroutine to be restarted at the top of the function.

It is not valid to use this command on an active coroutine or coroutine 0. If you do so, a runtime error will be triggered.
