PERFORM CHECKLIST FOR INTERFACES


Syntax

PERFORM CHECKLIST FOR INTERFACES

Description

Puts details of all interfaces into the checklist.

When this command is run, the checklist is populated with the IP addresses of the local network interfaces and other details relevant to those interfaces.

String = The device name of the interface
Value A = The interface IP address
Value B = The network mask for the interface
Value C = The IP address of the gateway for the interface
Value D = The IP address of the DHCP server (if applicable)

An alternative way to get interface IP addresses is to use the INTERFACE COUNT( ) and INTERFACE IP( ), but less information is available using that method.
