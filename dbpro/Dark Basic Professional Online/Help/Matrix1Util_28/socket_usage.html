SOCKET USAGE


Syntax

Usage = SOCKET USAGE ( Handle )

Description

Determines the current usage of the socket.

It returns 0 for an unassigned socket, 1 for a listening socket, and 2 for a data socket. UDP sockets always register as data sockets.
A value of -1 is returned if the Handle is not a valid socket.
