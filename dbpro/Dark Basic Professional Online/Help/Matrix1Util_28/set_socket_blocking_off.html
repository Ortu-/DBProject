SET SOCKET BLOCKING OFF


Syntax

SET SOCKET BLOCKING OFF Handle

Description

Switch the socket to non-blocking mode. It can be used on a socket at any time.

If used on a listening socket, this setting will be inherited by any socket created by the ACCEPT SOCKET function.
