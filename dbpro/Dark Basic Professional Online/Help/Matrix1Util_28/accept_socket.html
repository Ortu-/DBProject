ACCEPT SOCKET


Syntax

Handle = ACCEPT SOCKET ( Listening Handle )
Handle = ACCEPT SOCKET ( Listening Handle, Timeout )

Description

This function permits an incoming connection attempt on a socket. You should only use this command on a socket that has been set to listening mode using the LISTEN SOCKET command.

Returns a new handle connected to the remote machine, ready for reading and writing.
A zero is returned on a failure.
