SAVE LOOKUP TO INI


Syntax

SAVE LOOKUP TO INI Lookup id, Ini filename$

Description

This command can be used to write an Ini file from the lookup table specified.

The last slash within the key ( / ) is used as a separator to determine the section name and the ini key - for example 'abc/123/qwerty' will have a section name of 'abc/123' and an ini key of 'qwerty'.

If there is no slash within the name, the whole key is used for the section name, with the ini key kept as an empty string.
