APPEND LIMB


Syntax

Limb number = APPEND LIMB( Object number, Mesh number )
Limb number = APPEND LIMB( Object number, Mesh number, Limb name )
APPEND LIMB Object number, Mesh number
APPEND LIMB Object number, Mesh number, Limb name

Description

These commands and functions behave in the same way as the ADD LIMB command except they do not require you to specify a limb number.

The first two forms of this command will add a new limb to the object specified, and return the new limb number. The only difference between the two commands is that the second will allow you to name the limb as it is created.

The third and fourth forms of this command will do the same as the functions, except that they do not return the new limb number.
