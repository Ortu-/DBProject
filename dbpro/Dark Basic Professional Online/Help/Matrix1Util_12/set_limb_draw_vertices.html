SET LIMB DRAW VERTICES


Syntax

Primitives = SET LIMB DRAW VERTICES Object number, Limb number, Number of vertices

Description

This command allows you to control the number of vertices that will be processed for the limb.

The number specified must be between 0 and the allowable maximum for that limb (set at limb creation time) as determined by using the GET LIMB MAX DRAW VERTICES function.
Care should be taken when using this function that all vertices that are required to be rendered are correctly placed at the beginning of the vertex buffer, and all indexes (if used) refer to only these vertices.
