SET LIMB GHOST ON


Syntax

SET LIMB GHOST ON Object number, Limb number
SET LIMB GHOST ON Object number, Limb number, Ghost mode

Description

This command is used to enable ghosting on the limb specified. The Ghost mode can be a number between 0 and 5 as per the standard GHOST OBJECT ON command.
If the Ghost mode is not specified, the ghost mode will be set to 0.
