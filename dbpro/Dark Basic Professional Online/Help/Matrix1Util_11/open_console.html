OPEN CONSOLE


Syntax

OPEN CONSOLE
OPEN CONSOLE Width, Height
OPEN CONSOLE Width, Height, Buffer Height
OPEN CONSOLE WindowTitle
OPEN CONSOLE WindowTitle, Width, Height
OPEN CONSOLE WindowTitle, Width, Height, Buffer Height

Description

This command is used to open the console window.

Where specified, the title will be applied to the console window title bar.
The Width and Height parameters can be used to generate a console window of the specified size, however you will be restricted to the size that your system can display.
The Buffer Height parameter, if greater than the Height of the window, will cause a scrollbar to appear on the right of the window, allowing you to scroll back and forth to see what has been displayed. If the Buffer Height is less than the Height, then the parameter is ignored.

If the console window is already open, the command will be silently ignored.

Where not specified, the console size will be 80 x 25, with a buffer of 500 lines.
