Matrix1 Utility Plug-ins

Matrix1Util_12

This plug-in provides limb information, including limb relationships.
It allows you to set a limbs name, and to search for limbs by name.

There are low-level commands within this plug-in that also allow you to control transparency and multi-texturing at the limb level.
