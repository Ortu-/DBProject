DirLight_SetVisible


Syntax

DirLight_SetVisible(boolean SetVisible)

  • [boolean SetVisible] - A value of 0 will hide the directional light and lens flare, a value of 1 will show them.

Description

This command will set the visible status of the directional light.

This value is stored internally as the global variable: boolean AL_DirLightVisible

When hiding the directional light, the object referenced by AL_DirLightObject is excluded and hidden, and the object referenced by AL_AmbientObject is included and shown. The lensflare is hidden through a call to LensFlare_SetVisible(i LensFlareIndex, i Value)

When showing the directional light, the object referenced by AL_DirLightObject is included and shown, and the object referenced by AL_AmbientObject is excluded and hidden. The lensflare is shown through a call to LensFlare_SetVisible(i LensFlareIndex, i Value)

BUG WARNING:There is a bug in the default 20150325 distribution of Advanced Lighting in this command.

When showing the directional light via DirLight_SetVisible(1), the call to LensFlare_SetVisible(i LensFlareIndex, i Value) incorrectly uses a value of 0 (hide) this must be changed to 1 in order to correctly restore the lensflare when using DirLight_SetVisible(1).

This occurs on line 94 of .\Advanced Lighting\Include\Advanced Lighting\AL_Directional.dba

if AL_DirLightLensFlare.x>-1 then LensFlare_SetVisible(AL_DirLightLensFlare.x,0)

Replace with:

if AL_DirLightLensFlare.x>-1 then LensFlare_SetVisible(AL_DirLightLensFlare.x,1)


Samples

DirLight_SetVisible(1)
