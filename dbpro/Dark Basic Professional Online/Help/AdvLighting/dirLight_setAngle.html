Advanced Lighting


DirLight_SetAngle


Syntax

DirLight_SetAngle(float AngleX, float AngleY, float AngleZ)

  • [float AngleX] - X axis angle in the range of 0.0 - 360.0
  • [float AngleY] - Y axis angle in the range of 0.0 - 360.0
  • [float AngleZ] - Z axis angle in the range of 0.0 - 360.0

Description

This command will set the direction of the directional light through its xyz angle components.

These angle values are stored internally as the global variable:
float AL_DirLightAngle.x
float AL_DirLightAngle.y
float AL_DirLightAngle.z

This change will also affect the internal global variable:
AL_DirLightDirection.x
AL_DirLightDirection.y
AL_DirLightDirection.z

This command will set the effect constant 'LightDirection' of the current .\Advanced Lighting\Shaders\Lighting\Directional\*.fx shader
00.fx will be used if specular is disabled and ssao is disabled on the directional light.
01.fx will be used if specular is disabled and ssao is enabled on the directional light.
10.fx will be used if specular is enabled and ssao is disabled on the directional light.
11.fx will be used if specular is enabled and ssao is enabled on the directional light.
The global variable AL_DirLightEffect provides a reference to the effect number of this shader.

This command will also set the effect constant 'LightDirection' in all effects registered to the AL_Effect() and AL_AlphaEffect() arrays.

This command will set the effect constant 'LightDirection' in the scattering effect registered to AL_PostProcessingEffect(3)

This command will set the effect constants 'LightDir1' and 'LightDir2' in the cloud effect registered to AL_CloudsEffect(1)

This command will set the Water_AreaFog caustics direction.

This command will set the effect constant 'SpecularColor' in all effects registered to the Water_Effect() array.


Samples

DirLight_SetAngle(35.0, 40.0, 0.0)
