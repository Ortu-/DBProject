Advanced Lighting


DirLight_Create


Syntax

DirLight_Create(boolean UseSpecular, boolean UseSSAO, integer ShadowQuality, integer ShadowMapSize)

  • [boolean UseSpecular] - A value of 0 will disable specular lighting on the created light, 1 will enable it.
  • [boolean UseSSAO] - A value of 0 will disable sub-surface lighting on the created light, 1 will enable it.
  • [integer ShadowQuality] - A value of 0 will disable shadows from being cast by the light, 1 will enable pixelated blocky shadows, and 2 will enable smooth anti-aliased shadows.
  • [integer ShadowMapSize] - This value should be a square power of 2 and will determine the size of the shadow map texture image. This will affect the sharpness quality of the shadows cast by the light. A value of 256 will give fuzzy, low quality shadows, 512 will give average quality, and 1024 will give sharp, high quality shadows.

Description

This command will set up the directional light within the scene.
A directional light has no physical position and is used to simulate the sun or other global light source.
There may only be one directional light.

The shadow properties can be altered after the light is created through a call to DirLight_SetShadow(i ShadowQuality, i ShadowMapSize)

This command will load the .\Advanced Lighting\Shaders\Lighting\Directional\*.fx shader
00.fx will be used if specular is disabled and ssao is disabled on the directional light.
01.fx will be used if specular is disabled and ssao is enabled on the directional light.
10.fx will be used if specular is enabled and ssao is disabled on the directional light.
11.fx will be used if specular is enabled and ssao is enabled on the directional light.
The global variable AL_DirLightEffect provides a reference to the effect number of this shader.


Samples

DirLight_Create(1, 0, 2, 512)
