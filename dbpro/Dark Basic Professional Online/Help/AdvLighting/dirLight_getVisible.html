Advanced Lighting


DirLight_GetVisible


Syntax

Return float Visible = DirLight_GetVisible(null)

Description

Returns visible status of the directional light.

This value is stored internally as the global variable: boolean AL_DirLightVisible

The internal variable is of type boolean, but is cast as a float for the function return.


Samples

isVisible = DirLight_GetVisible()
