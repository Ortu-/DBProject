DirLight_SetSpecularPower


Syntax

DirLight_SetSpecularPower(float Power)

  • [float Power] is expected in the range of 0 - 256, default is set to 256.

Description

This command will set the effect constant 'SpecularPower' of the current .\Advanced Lighting\Shaders\Lighting\Directional\*.fx shader

10.fx will be used if specular is enabled and ssao is disabled on the directional light.

11.fx will be used if specular is enabled and ssao is enabled on the directional light.

The global variable AL_DirLightEffect provides a reference to the effect number of this shader.


Samples

DirLight_SetSpecularPower(128.0)
