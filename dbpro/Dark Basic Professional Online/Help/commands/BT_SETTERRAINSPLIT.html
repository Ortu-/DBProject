BT SetTerrainSplit


Syntax

BT SetTerrainSplit(TerrainID,Split)

Description

This command splits terrains down into small chunks. This is so the system can cull parts of the terrain individually and do LOD.

Parameters

TerrainID (integer) - This is the identifier of the terrain you want to set the split of. This value is created in the BT MakeTerrain function.
Split (integer) - This is the amount of sectors you want on each side. For example, set this to 4 and it will split the terrain into a 4x4 grid of sectors.

Returns

Nothing.
