BT SetTerrainYScale


Syntax

BT SetTerrainYScale(TerrainID,Scale)

Description

Sets the Y scale of the terrain. When set to 1 the terrain will be the same height as the pixel colours on the heightmap. Setting this to 2 will make the terrain double in height. To increase the size of a terrain see BT_SetTerrainScale.

Parameters

Terrain (integer) - This is the identifier of the terrain you want to set the texture to. This value is created in the BT MakeTerrain function.
Scale (float) - This is the scale value.

Returns

Nothing.
