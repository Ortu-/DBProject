BT LoadTerrain


Syntax

TerrainID = BT LoadTerrain(FileName,ObjectID)
TerrainID = BT LoadTerrain(FileName,ObjectID,BuildAll)

Description

This loads a terrain from a file. The file must have been previously saved with BT SaveTerrain. Terrain files in GDK are the exact same format as terrains in DBPro.

Parameters

FileName (string) - The filename of the file.
ObjectID (integer) - The DBPro ObjectID to set to this terrain.
BuildAll (boolean) - Set this to 1 and the terrain will be loaded and built without needing a build loop.

Returns

Terrain ID (integer) - Use this value in other functions to point to this terrain.
