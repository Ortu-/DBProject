BT BuildTerrain


Syntax

BT BuildTerrain(TerrainID,TerrainObjectID)
BT BuildTerrain(TerrainID,TerrainObjectID,GenerateTerrain)

Description

This command builds the terrain. Its processes the heightmap and parameters and generates vertexdata for the terrain. Use BT ContinueBuild or set the GenerateTerrain flag to turn the vertexdata into visible objects.

Parameters

TerrainID (integer) - This is the unique Identifier of the terrain you want to build.
TerrainObjectID (integer) - Set this to a free DBPro Object ID which the terrain will use.
GenerateTerrain (integer) - (optional) Set this to 1 to make BlitzTerrain to generate the whole terrain without a build loop. If this is set, you will beable to use and render the terrain immediately after this function has returned. Default value is 0.

Returns

Nothing.
