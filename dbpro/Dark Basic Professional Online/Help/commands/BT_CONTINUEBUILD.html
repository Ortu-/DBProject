BT ContinueBuild


Syntax

Progress=BT ContinueBuild()

Description

This command should be used after the BT BuildTerrain function and keep being called until it returns -1.

Parameters

No Parameters

Returns

Progress. This command will return the percentage of which the terrain engine has done building the terrain when the terrain has finnished building it will return -1.
