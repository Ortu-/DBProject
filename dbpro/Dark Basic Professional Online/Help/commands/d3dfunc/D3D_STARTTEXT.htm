D3D_STARTTEXT

This command is used to begin antialiased text drawing operations. This must be called before any call to D3D_TEXT or D3D_BOXTEXT.
NOTE: You should not perform any other 2D operations while this command is in effect in order to recieve the correct results.

SYNTAX

D3D_STARTTEXT
