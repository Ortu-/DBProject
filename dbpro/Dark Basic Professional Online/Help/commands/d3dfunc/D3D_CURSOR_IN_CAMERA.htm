D3D_CURSOR_IN_CAMERA

This command is used to determine which camera's viewport the cursor is currently located within. If the cursor is within a camera viewport the camera number is returned, otherwise a value of -1 is returned.

SYNTAX

Return Integer = D3D_CURSOR_IN_CAMERA First Camera, Last Camera
