D3D_GET_CAMERA_UP

This command is used to calculate the camera's upward facing vector. If you do not supply an optional camera number the current camera will be used.

Use the D3D_VECTOR_X,  D3D_VECTOR_Y, and D3D_VECTOR_Z to retrieve the pick vectors.

SYNTAX

D3D_GET_CAMERA_UP [Camera]
