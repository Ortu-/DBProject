D3D_BATCH_DRAW_DOT2D

This command will draw all dots in the current batch. When the clear flag is set to anything but 0, then the current batch will be
deleted after use, and any further use will require reinitialisation using D3D_BATCH_SET_DOT2D.

SYNTAX

D3D_BATCH_DRAW_DOT2D Clear Flag
