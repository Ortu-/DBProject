D3D_SET_ORTHO_PROJECTION

This command is used to create a left-handed orthographic projection camera. If you do not supply an optional camera number the current camera will be used. View width is specified in DBPro units.

SYNTAX

D3D_SET_ORTHO_PROJECTION [Camera,] View Width
