ADD VECTOR3 TO CAMERA POSITION


Syntax

ADD VECTOR3 TO CAMERA POSITION Vector3, CameraId
ADD VECTOR3 TO CAMERA POSITION Vector3, VectorScale, CameraId

Description

This command can be used to adjust the position of the camera by the vector.

The first form of this command simply adds the vector to the camera position.

The second form of this command multiplies the vector by the scale provided and then adds the result to the camera position.
