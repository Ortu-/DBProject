Matrix1 Utility Plug-ins

Matrix1Util_05

This plug-in provides you with the functions required to locate a free resource id (eg image, or object etc) from the standard DBPro commands, avoiding the need to track free id's in your code.

Equivalent functions for the Matrix1Util plug-ins can be found in each plug-in that has resources to track (i.e. Look in the Banks plug-in for FIND FREE BANK).
